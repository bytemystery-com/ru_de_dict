# ru_de_dict
Russian -> German dictionary.
Base format is XDXF. I also created with [PsyGlosdsary](https://github.com/ilius/pyglossary) a dict version
for my PocketReader. As data source I used the [OpenRussian Website] (https://de.openrussian.org/).
